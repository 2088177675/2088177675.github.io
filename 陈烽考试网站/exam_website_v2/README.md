## 功能特性

### 教师端

| 功能 | 描述 |
|------|------|
| 📝 考试管理 | 发布考试、编辑考试内容、删除考试、查看考试详情 |
| 🤖 AI 自动出题 | 基于 DeepSeek API 自动生成题目、选项、答案和解析 |
| 👨‍🎓 学生管理 | 添加学生、删除学生、按班级筛选学生 |
| 🏫 班级管理 | 添加班级、删除班级、查看班级学生人数 |
| 📈 成绩统计 | 查看所有学生成绩、按班级/考试筛选、查看正确率 |

### 学生端

| 功能 | 描述 |
|------|------|
| 📝 参加考试 | 在线答题、倒计时提醒、自动提交 |
| 📈 查看成绩 | 查看历史成绩、得分、正确率、用时 |
| ❌ 错题回顾 | 查看错题详情、正确答案、答案解析 |

### 系统特性

- **数据持久化**：使用浏览器 localStorage 存储数据，刷新不丢失
- **响应式设计**：适配桌面端和移动端
- **权限控制**：教师/学生独立页面，未登录自动跳转
- **实时倒计时**：考试过程带倒计时，时间到自动提交

## 使用指南

### 教师端

#### 1. 登录

- 选择 **"教师登录"** 标签
- 输入账号密码：`teacher` / `123456`
- 点击登录

#### 2. 发布考试（手动出题）

```
考试管理 → 发布新考试 → 填写考试信息 → 添加题目 → 保存考试
```

每道题目需要填写：
- 题目内容
- 4 个选项（A/B/C/D）
- 正确答案
- 分值
- [可选] 答案解析

#### 3. 发布考试（AI 自动出题）

```
考试管理 → 发布新考试 → 填写考试信息 → 🤖 AI 自动出题 → 输入主题 → 开始生成 → 保存考试
```

详见 [AI 自动出题](#ai-自动出题) 章节。

#### 4. 管理学生

```
学生管理 → 添加学生 → 填写姓名/账号/密码/班级 → 保存
```

#### 5. 管理班级

```
班级管理 → 添加班级 → 填写班级名称/描述 → 保存
```

#### 6. 查看成绩

```
成绩统计 → [可按班级/考试筛选] → 查看学生得分、正确率、用时
```

---

### 学生端

#### 1. 登录

- 选择 **"学生登录"** 标签
- 输入账号密码（如：`student1` / `123456`）
- **选择班级**（如：软件工程1班）
- 点击登录

#### 2. 参加考试

```
我的考试 → 点击"开始考试" → 答题 → 提交试卷
```

考试规则：
- 倒计时显示在右上角
- 点击选项即可选择答案
- 可随时切换题目
- 时间到自动提交
- 提交后显示得分

#### 3. 查看成绩

```
我的成绩 → 查看所有历史考试成绩
```

显示信息：
- 得分 / 满分
- 正确率
- 用时
- 提交时间

#### 4. 错题回顾

```
错题回顾 → 查看所有错题
```

每道错题显示：
- 你的答案（红色标记）
- 正确答案（绿色标记）
- 答案解析

## AI 自动出题

### 配置 API Key

1. 访问 [DeepSeek 开放平台](https://platform.deepseek.com)
2. 注册/登录账号
3. 左侧菜单 → **API Keys** → 创建 API Key
4. 复制 Key（格式：`sk-xxxxxxxxxxxxxxxx`）
5. 打开 `js/teacher-exams.js`，修改第 6 行：

```javascript
const DEEPSEEK_API_KEY = 'sk-你的真实API密钥';
```

### 使用步骤

```
1. 教师登录 → 考试管理 → 发布新考试
2. 填写考试基本信息（名称、班级、时长、总分）
3. 点击 "🤖 AI 自动出题（DeepSeek）" 按钮
4. 在弹窗中填写：
   - 出题主题：如 "JavaScript基础"、"中国古代史"、"微积分"
   - 学科分类：计算机 / 数学 / 英语 等
   - 题目数量：3 / 5 / 10 / 15 道
   - 难度：简单 / 中等 / 困难
5. 点击 "开始生成"
6. AI 自动生成题目、选项、正确答案和解析
7. 可手动编辑修改生成的题目
8. 点击 "保存考试"
```

### AI 生成内容示例

```json
[
  {
    "content": "JavaScript中，以下哪个不是基本数据类型？",
    "options": ["String", "Number", "Array", "Boolean"],
    "correctAnswer": 2,
    "score": 20,
    "explanation": "Array是引用类型，不是基本数据类型。"
  }
]
```

### 注意事项

| 问题 | 解决方案 |
|------|----------|
| 跨域错误（CORS） | 安装浏览器插件 "Allow CORS" 或 "CORS Unblock" |
| API Key 无效 | 检查 Key 是否正确，是否已充值 |
| 生成格式错误 | 重新点击生成，或检查网络连接 |
| 网络超时 | 检查网络连接，或稍后重试 |

## 常见问题

### Q1：数据会丢失吗？

**不会。** 所有数据存储在浏览器的 localStorage 中，只要不清理浏览器缓存，数据就会一直保留。如需重置数据，在浏览器控制台执行：

```javascript
localStorage.clear();
location.reload();
```

### Q2：如何添加更多演示数据？

修改 `js/data.js` 中的 `DataStore.init()` 方法，添加更多班级、学生、考试或成绩数据。

### Q3：如何修改默认密码？

在 `js/data.js` 的初始化数据中修改 `password` 字段，或在学生管理页面添加新学生时设置。

### Q4：AI 出题收费吗？

DeepSeek API 按 token 计费，新用户通常有免费额度。具体费用请参考 [DeepSeek 官方定价](https://platform.deepseek.com)。

### Q5：支持多选题吗？

当前版本仅支持单选题。如需多选题，需要修改题目数据结构和相关逻辑。

### Q6：如何部署到服务器？

由于是纯前端项目，可直接部署到任何静态服务器：

```bash
# 使用 Nginx / Apache / GitHub Pages / Vercel 等
# 只需上传所有文件即可
```

### Q7：退出登录后跳转到 404？

请确保项目文件结构完整，`index.html` 位于项目根目录。如使用旧版本，请下载最新版本。
