    16гҖҒиҪ¬д№үеӯ—з¬ҰпјҲжҳҫзӨәзү№ж®Ҡз¬ҰеҸ·пјү¶

    TIP

    еҰӮдҪ•еңЁ MarkDown ж–ҮжЎЈдёӯжү“еҮәзү№ж®Ҡеӯ—з¬ҰпјҹиҰҒжҳҫзӨәеҺҹжң¬з”ЁдәҺеңЁ Markdown ж–ҮжЎЈдёӯж јејҸеҢ–ж–Үжң¬зҡ„ж–Үеӯ—еӯ—з¬ҰпјҢеңЁеӯ—з¬ҰеүҚйқўж·»еҠ еҸҚж–ңжқ  ``гҖӮ

    17гҖҒиЎЁж ј¶

    TIP

    MarkDown д№ҰеҶҷиЎЁж јзҡ„е…·дҪ“иҜӯжі•

    иЎЁеӨҙ

    • иЎЁеӨҙз”ұдёҖиЎҢж–Үжң¬з»„жҲҗпјҢжҜҸдёӘеҚ•е…ғж јзҡ„еҶ…е®№з”Ё |з¬ҰеҸ·еҲҶйҡ”гҖӮ

    • иЎЁеӨҙдёӢж–№йңҖиҰҒдёҖиЎҢеҲҶйҡ”иЎҢпјҢз”ЁдәҺеҢәеҲҶиЎЁеӨҙе’ҢиЎЁж јеҶ…е®№гҖӮеҲҶйҡ”иЎҢз”ұ -з¬ҰеҸ·з»„жҲҗпјҢдё” -зҡ„ж•°йҮҸеә”дёҺиЎЁеӨҙдёӯзҡ„еҲ—ж•°зӣёеҢ№й…ҚпјҲдҪҶйҖҡеёёиҮіе°‘дёә 3 дёӘ -пјүгҖӮеҲҶйҡ”иЎҢзҡ„е·Ұдҫ§е’ҢеҸідҫ§еҸҜд»ҘзңҒз•Ҙ |з¬ҰеҸ·гҖӮ

    еҚ•е…ғж јеҶ…е®№

    • еҚ•е…ғж јеҶ…е®№д№ҹдҪҝз”Ё |з¬ҰеҸ·еҲҶйҡ”пјҢжҜҸиЎҢд»ЈиЎЁиЎЁж јдёӯзҡ„дёҖиЎҢж•°жҚ®гҖӮ

    • еҚ•е…ғж јеҶ…е®№еҸҜд»ҘеҢ…еҗ«ж–Үжң¬гҖҒж•°еӯ—гҖҒй“ҫжҺҘзӯүгҖӮ

    еҜ№йҪҗж–№ејҸ

    • й»ҳи®Өжғ…еҶөдёӢпјҢMarkdown иЎЁж јзҡ„еҚ•е…ғж јеҶ…е®№еұ…е·ҰеҜ№йҪҗгҖӮ

    • еҸҜд»ҘйҖҡиҝҮеңЁеҲҶйҡ”иЎҢдёӨдҫ§зҡ„ :з¬ҰеҸ·жқҘи°ғж•ҙеҜ№йҪҗж–№ејҸгҖӮдҫӢеҰӮпјҢ:--:иЎЁзӨәеұ…дёӯеҜ№йҪҗпјҢ:--иЎЁзӨәе·ҰеҜ№йҪҗпјҲе®һйҷ…дёҠжҳҜй»ҳи®ӨеҜ№йҪҗж–№ејҸпјҢеӣ жӯӨйҖҡеёёдёҚйңҖиҰҒжҳҫејҸжҢҮе®ҡпјүпјҢ--:иЎЁзӨәеҸіеҜ№йҪҗгҖӮ
